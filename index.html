My Work

Micro-Budget Latent Diffusion Transformer for High-Resolution Image Synthesis

Developed a 1.2B-parameter Latent Diffusion Transformer from scratch in PyTorch, achieving a state-of-the-art 12.7 FID score on the COCO dataset at a cost of $1,314. Engineered a progressive pretraining pipeline with dynamic positional embeddings and a lightweight PatchMixer, enabling efficient masked and unmasked denoising. Scaled training using Fully Sharded Data Parallel (FSDP) and Composer Hydra for optimal performance.

Transformer based Autoregressive Image generation using VQGAN

Developed a generative model in PyTorch, combining VQGAN (108M params) for discrete neural representation learning and GPT-2 (91.5M params) for autoregressive image generation. Implemented a Vector Quantized VAE with adversarial training, multi-objective loss, and gradient flow reparameterization to learn a structured latent space. Trained on the Oxford Flowers dataset using BFloat16 mixed precision, achieving high-fidelity image synthesis.

Foundation Model for Text Completion

Developed a 124M-parameter GPT-2 model from scratch, trained on 10B tokens from the FineWebEdu dataset, achieving a validation loss of 3.1440 and HellaSwag score of 30.22%. Scaled training to 50B tokens, matching OpenAI-GPT3's performance, and optimized using PyTorch DDP, BFloat16 mixed precision, and flash attention on an 8x A100 cluster.

U-Net for Image Reconstruction

Implemented a U-Net architecture in PyTorch with a 4-stage encoder-decoder structure, achieving 16x compression in latent space (256x256 → 16x16). Designed with self-attention, skip connections, and transposed convolutions for realistic RGB image generation. Optimized training using BFloat16 mixed precision, reducing memory usage by 50% and enabling larger batch sizes on A100 GPUs.
